# Repositório para estudos de assembly.

OK, você deve estar se perguntando o motivo disso...
  - Entender melhor como o computador funciona
  - Entender melhor como o sistema operacional funciona
  - Entender meios de modificar um binário sem ter o fonte
  - Aprender sobre confecção de exploits e shellcodes
  - Aprender sobre engenharia reversa
  - Encontrar falhas em softwares
  - Participar de CTF e não passar (muita) vergonha

## Dependências
  - yasm
  - binutils (para obter o ld)
  - lld (extensão para ld x86-64)
  - Linux x86-64 (um termux no Android também funciona)
